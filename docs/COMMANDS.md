# 飞书机器人命令使用指南

本文档介绍飞书机器人支持的命令及其用法。

## 命令列表

### 帮助命令
- 命令：`/帮助` 或 `/help`
- 功能：显示帮助信息

### 日报汇总
- 命令：`/日报汇总 [日期]`
- 功能：生成指定日期的日报汇总，默认今天
- 示例：
  - `/日报汇总`
  - `/日报汇总 2026-02-26`

### 我的日报
- 命令：`/我的日报 [日期]`
- 功能：查询自己的日报
- 日期支持：`今天`、`昨天`、`YYYY-MM-DD`

### 设置调休
- 命令：`/设置调休 <姓名> [日期]`
- 功能：设置某人调休，默认今天
- 示例：`/设置调休 张三 2026-02-26`

### 取消调休
- 命令：`/取消调休 <姓名> [日期]`
- 功能：取消调休设置

### 查询调休
- 命令：`/查询调休 [日期]`
- 功能：查询指定日期调休人员

## 常见问题

### 命令不生效怎么办？
1. 确认命令以 `/` 开头
2. 确认格式正确
3. 使用 `/帮助` 查看说明

### 日期格式是什么？
统一使用 `YYYY-MM-DD`，例如 `2026-02-26`。

### 谁可以使用这些命令？
默认群成员都可使用基础命令；如需限制可结合 `COMMAND_ADMIN_USERS` 扩展鉴权。
